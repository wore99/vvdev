LingoSynth - Support

Application Overview

LingoSynth is an innovative language learning video player designed for language learners who want to improve their listening comprehension skills.

Key Features

  • Smart Playback Control: Easily pause to get previous sentence content and explanation
  • No-Subtitle Mode: Help users break free from subtitle dependency
  • Real-time Language Assistance: Click to view explanations of difficult sentences
  • Support for Multiple Video Formats
  • Local Video Import Function

Frequently Asked Questions

How to Import Videos?

You can import videos through the following methods:

  • Import from Files app
  • Share via AirDrop
  • Share from other supported apps

How to Use No-Subtitle Mode?

When playing a video, no-subtitle mode is enabled by default. When you encounter content you don't understand, simply click the pause button, and the previous sentence's content and explanation will be displayed.

Which Languages are Supported?

Currently, we support learning major languages including English, Japanese, Korean, and more. We will continue to add support for more languages.

Is Background Playback Supported?

Yes, you can continue listening to audio while the screen is locked or while using other apps.
